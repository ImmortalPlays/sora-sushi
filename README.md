# Sora Sushi — Sample Restaurant Site

A single-page sample site for a modern Japanese restaurant, with a working
front-end ordering cart. Built as a portfolio demonstration.

## Features
- Responsive single-page layout (Menu, About, Gallery, Hours & Location)
- Slide-in **cart** with quantities and a live total (pure JavaScript)
- No build step — just HTML, CSS, and JS

## Run it locally
Open `index.html` in any browser. That's it.

## Structure
| File | What it does |
|------|--------------|
| `index.html` | Page content and structure |
| `styles.css` | All styling and the design system |
| `script.js`  | Menu data + cart logic |
| `images/`    | Drop real photos here |

## Notes
The cart is a front-end demo — checkout does not process payment. To take real
orders, embed a service like Square Online or Toast.
