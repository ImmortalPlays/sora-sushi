# Menu photos

Drop a photo for each dish here, named by its **id** from `script.js`, e.g.:

- `salmon-nigiri.jpg`
- `dragon-roll.jpg`
- `tonkotsu.jpg`  … etc.

On desktop, hovering a menu card slides the photo up with a faded info bar.
If a photo is missing, the card just shows a colored fallback (no error).
Keep each image roughly square and under ~300 KB.
